# Project Context
## 2026 미래 성장 산업 & 주식 투자 리서치

## 1. 목적

이 프로젝트는 장기 투자 관점에서 미래 성장 산업과 주요 글로벌 상장기업을 분석하는 단일 HTML 리포트다. 종목 추천보다 산업 구조, 기술 변화, 시장 규모, 기업별 실행력, 밸류에이션 부담, 리스크를 함께 보는 의사결정 프레임을 제공한다.

## 2. 독자

| 독자 | 필요한 정보 |
|---|---|
| 장기 개인 투자자 | 산업 구조, 핵심 기업, 리스크, 분산 기준 |
| 테마 투자자 | AI 인프라, 반도체, 전력, 로봇, 헬스케어 AI 등 비교 |
| 초보 리서처 | 어떤 데이터를 어디서 확인해야 하는지 |
| 모바일 독자 | 한 화면에서 핵심 카드와 표를 빠르게 확인 |

## 3. 의사결정 기준

이 리포트는 다음 질문에 답하는 방식으로 구성한다.

1. 이 산업은 구조적으로 커지고 있는가?
2. 성장의 병목을 가진 기업은 누구인가?
3. 매출 성장과 현금흐름이 밸류에이션을 정당화하는가?
4. CAPEX, 금리, 규제, 공급망 리스크가 어느 정도인가?
5. 어떤 조건에서 투자 매력도가 높아지거나 낮아지는가?

## 6. 버전 관리

- `index.html`은 GitHub Pages에서 실제로 열리는 메인 파일이다.
- `investment-research-2026.html`은 동일 리포트의 백업/원본 파일이다.
- 두 HTML 파일은 내용이 같아야 한다.
- 수치 업데이트 시 기준일과 Sources 섹션을 함께 갱신한다.
